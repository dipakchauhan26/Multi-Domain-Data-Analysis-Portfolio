# Multi-Domain Data Analysis Portfolio

## Overview
This project is a complete data analysis portfolio consisting of 5 real-world projects across different domains.

The portfolio demonstrates:
- Data Cleaning
- Data Analysis
- Data Visualization
- Statistical Analysis
- Business Insights


##  Projects Included

###  Project 1: Supermarket Sales Analysis
- Sales trends
- Product performance
- Customer behavior

###  Project 2: Student Performance Analysis
- Pass/Fail rates
- Attendance impact

###  Project 3: Weather Data Analysis
- Temperature trends
- Seasonal patterns

###  Project 4: Healthcare Analysis
- Disease trends
- Recovery rates

###  Project 5: Financial Analysis
- Stock trends
- Market analysis


##  Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- SciPy
- ReportLab


##  Folder Structure

- data/ → datasets  
- notebooks/ → Jupyter notebooks  
- reports/ → PDF reports  
- src/ → reusable code  
- visualizations/ → charts  
- presentation/ → PPT  
- docs/ → documentation
