#  Healthcare Data Analysis

##  Objective
Analyze healthcare data to understand disease trends and recovery rates.

##  Dataset Description
The dataset includes:
- Patient data
- Disease cases
- Recovery records
- Regional data

##  Data Cleaning
- Removed duplicates
- Handled missing values

##  Analysis Performed
- Case trend analysis
- Recovery rate analysis
- Region-wise analysis

##  Visualizations
- Line chart (case trends)
- Bar chart (region analysis)
- Heatmap (correlation)

##  Key Insights
- Cases increase during certain periods
- Recovery rates are improving
- Some regions are more affected

##  Recommendations
- Improve healthcare facilities
- Focus on high-risk areas
- Increase awareness programs

##  Conclusion
Healthcare data helps in better planning and resource allocation.
