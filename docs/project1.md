# Supermarket Sales Analysis

## Objective
Analyze supermarket sales data to identify trends, best-selling products, and customer behavior.

##  Dataset Description
The dataset contains:
- Product categories
- Customer types
- Sales transactions
- Quantity and pricing details
- Payment methods

##  Data Cleaning
- Removed duplicate records
- Handled missing values
- Created Revenue column (Quantity × Price)

##  Analysis Performed
- Daily sales trend analysis
- Product performance analysis
- Customer behavior analysis
- Payment method analysis

##  Visualizations
- Line chart (sales trend)
- Bar chart (category performance)
- Histogram (sales distribution)
- Heatmap (correlation)

##  Key Insights
- Electronics is the top-selling category
- Weekend sales are higher
- Evening hours generate maximum revenue

##  Recommendations
- Increase stock for high-demand products
- Run promotions during weekends
- Focus on peak hours

##  Conclusion
This analysis helps improve business decisions using data-driven insights.
