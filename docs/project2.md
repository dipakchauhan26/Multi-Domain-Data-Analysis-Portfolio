#  Student Performance Analysis

##  Objective
Analyze student performance to understand pass/fail rates and impact of attendance.

##  Dataset Description
The dataset includes:
- Student marks
- Subject details
- Attendance percentage

##  Data Cleaning
- Removed duplicates
- Handled missing values
- Created Result column (Pass/Fail)

##  Analysis Performed
- Pass/Fail rate calculation
- Subject-wise performance
- Attendance vs marks analysis

##  Visualizations
- Histogram (marks distribution)
- Bar chart (subject performance)
- Scatter plot (attendance vs marks)

##  Key Insights
- Higher attendance leads to better performance
- Some subjects have lower scores
- Most students are average performers

##  Recommendations
- Improve attendance policies
- Provide extra classes for weak subjects
- Monitor student performance

##  Conclusion
Attendance plays a key role in academic success.
