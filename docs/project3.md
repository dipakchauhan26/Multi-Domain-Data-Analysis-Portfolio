#  Weather Data Analysis

##  Objective
Analyze weather data to identify temperature trends, seasonal patterns, and rainfall distribution.

##  Dataset Description
The dataset includes:
- Temperature data
- Date and time
- Rainfall data
- Environmental variables

##  Data Cleaning
- Removed duplicates
- Handled missing values
- Converted date column
- Extracted month feature

##  Analysis Performed
- Temperature trend analysis
- Seasonal (monthly) analysis
- Rainfall distribution
- Extreme weather detection

##  Visualizations
- Line chart (temperature trend)
- Bar chart (monthly temperature)
- Histogram (rainfall)
- Heatmap (correlation)

##  Key Insights
- Temperature shows seasonal variation
- Rainfall occurs in specific months
- Weather patterns are cyclical

##  Recommendations
- Use data for agricultural planning
- Monitor extreme weather events
- Improve forecasting systems

##  Conclusion
Weather data analysis helps in planning and decision-making.
