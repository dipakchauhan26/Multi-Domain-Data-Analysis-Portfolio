#  Financial Data Analysis

##  Objective
Analyze financial data to understand stock trends and market behavior.

##  Dataset Description
The dataset includes:
- Stock prices
- Dates
- Volume traded

##  Data Cleaning
- Removed duplicates
- Handled missing values

##  Analysis Performed
- Stock trend analysis
- Moving average analysis
- Volatility analysis

##  Visualizations
- Line chart (price trend)
- Moving average chart
- Histogram (returns)

##  Key Insights
- Stock prices show trends over time
- Volatility varies across periods
- Market reacts to external factors

##  Recommendations
- Invest during stable trends
- Avoid high volatility periods
- Diversify investments

##  Conclusion
Financial analysis supports better investment decisions.
